Any file type can be linked to on the web.

HTML is made up of

  • text content (what you see)
  • markup (what it looks like)
  • references to other documents (eg. images and videos)

Markup looks like

  • < name>Contents< /name>
  • = < opening tag>Contents< /closing tag>
    • This is called an 'element'

Tags (make sure to close *most of* these!)

Inline Tags

  • < b>Bold< /b>
  • < em>emphasis< /em>
  • < a href="link">anchor links< /a>
  • < img src="url" alt="text">
    • - called a VOID tag
  • < br>
  • < span class="foo"> text < /span>
  • < strong>bold< /strong>

Block Tags

block text, can have height and width
  • < p>content< /p>
  • < div class="bar">text< /div>
  • < form>???< /form>
